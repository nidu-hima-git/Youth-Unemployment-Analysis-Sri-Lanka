# Youth-Unemployment-Analysis-Sri-Lanka
Statistical analysis and logistic regression modelling of youth unemployment patterns in Sri Lanka using Labour Force Survey 2023 data.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Analysis Included
- Data preprocessing
- Exploratory data analysis
- Chi-square tests
- Logistic regression
- Weighted logistic regression for class imbalance
